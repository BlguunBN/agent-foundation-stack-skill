| Research, scrape, summarize sources | `source-collection-agent` | `web`, `browser`, `file` |
| Jobs, internships, applications | `job-search-apply-agent` | `web`, `file`, `terminal` |
| Verify a stack or workflow | `agent-stack-verification-loop` or `personal-spec-kit-foundation` | `terminal`, `file`, `todo` |
| Motion / animation prompts | `scroll-animation-prompt-agent` | `image_gen`, `file` |
| Audio / narration | `fish-audio-agent` | `tts`, `file` |
| General execution on codebases | foundation trio + task-specific skill | `terminal`, `file`, `delegation` |
