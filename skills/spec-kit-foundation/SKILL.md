# Spec Kit Foundation

Use this skill when you want the agent to work spec-first instead of prompt-first.

## Flow
1. Write the spec/constitution.
2. Define the feature.
3. Build the implementation plan.
4. Break work into tasks.
5. Implement the tasks.
6. Analyze cross-artifact consistency.

## When to use
- New features with ambiguous scope
- Multi-step implementation work
- Any task that benefits from a spec → plan → tasks → implement sequence
