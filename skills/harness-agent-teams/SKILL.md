# Harness Agent Teams

Use this skill when a task should be split into specialized agents instead of handled by one general agent.

## Team patterns
- Pipeline
- Fan-out / Fan-in
- Expert Pool
- Producer-Reviewer
- Supervisor
- Hierarchical Delegation

## When to use
- Parallel independent work
- Clear subproblem boundaries
- Review-heavy workflows
- Long tasks that benefit from role separation
